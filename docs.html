Introdução

Bem-vindo à documentação completa do Design System. Este guia fornece informações detalhadas sobre como usar cada componente, variáveis CSS disponíveis e melhores práticas de implementação.

Como Começar

Para usar o Design System em seu projeto, inclua os seguintes arquivos CSS:

<!-- Variáveis CSS (obrigatório) -->
<link rel="stylesheet" href="css/variables.css">

<!-- Estilos dos componentes (obrigatório) -->
<link rel="stylesheet" href="css/styles.css">

<!-- Estilos auxiliares (opcional) -->
<link rel="stylesheet" href="css/index.css">

Estrutura de Arquivos

  • variables.css - Contém todas as variáveis CSS (cores, espaçamentos, tipografia)
  • styles.css - Contém os estilos dos componentes do design system
  • index.css - Contém estilos auxiliares para demonstração e layout

Variáveis CSS

Todas as variáveis CSS estão definidas no arquivo css/variables.css e podem ser facilmente customizadas para atender às necessidades do seu projeto.

Cores

Uso de Cores

As cores do design system devem ser usadas através das variáveis CSS para garantir consistência e facilitar manutenção.

Exemplo de Uso

/* Usando variáveis de cor */
.meu-elemento {
    background-color: var(--color-primary);
    color: var(--color-white);
    border: 1px solid var(--color-grey);
}

Tipografia

Hierarquia Tipográfica

O design system define uma hierarquia tipográfica clara com 5 níveis: H1, H2, H3, H4 e Paragraph.

Exemplo de Uso

<!-- HTML -->
<h1>Título Principal</h1>
<h2>Subtítulo Principal</h2>
<h3>Subtítulo Secundário</h3>
<h4>Subtítulo Terciário</h4>
<p>Texto de parágrafo</p>

Heading 1 - Título Principal (48px/60px)

Heading 2 - Subtítulo Principal (34px/50px)

Heading 3 - Subtítulo Secundário (28px/38px)

Heading 4 - Subtítulo Terciário (18px/22px)

Paragraph - Texto de parágrafo (14px/24px)

Espaçamento

Sistema de Espaçamento

O design system utiliza um sistema de espaçamento consistente com 6 tamanhos: XS, S, M, L, XL e XXL.

Exemplo de Uso

/* Usando variáveis de espaçamento */
.meu-elemento {
    padding: var(--spacing-m);
    margin-bottom: var(--spacing-l);
    gap: var(--spacing-s);
}
Tamanho Variável Valor Uso Recomendado
XS --spacing-xs 5px Espaçamento mínimo entre elementos muito próximos
S --spacing-s 10px Padding interno de elementos pequenos
M --spacing-m 15px Espaçamento padrão entre elementos relacionados
L --spacing-l 25px Espaçamento entre seções relacionadas
XL --spacing-xl 40px Espaçamento entre seções distintas
XXL --spacing-xxl 65px Espaçamento entre blocos principais de conteúdo

Botões

Botão Primary

Obrigatório

Botão principal usado para ações primárias. Utiliza a cor Primary do design system.

Exemplo de Uso

<!-- Botão Primary -->
<button type="button" class="btn btn--primary">
    Primary Button
</button>

<!-- Botão Primary Disabled -->
<button type="button" class="btn btn--primary" disabled>
    Primary Disabled
</button>

Classes CSS

  • .btn - Classe base do botão
  • .btn--primary - Modificador para botão primary

Botão Alternative

Opcional

Botão alternativo usado para ações secundárias. Utiliza a cor Secondary do design system com borda.

Exemplo de Uso

<!-- Botão Alternative -->
<button type="button" class="btn btn--alternative">
    Alternative Button
</button>

<!-- Botão Alternative Disabled -->
<button type="button" class="btn btn--alternative" disabled>
    Alternative Disabled
</button>

Classes CSS

  • .btn - Classe base do botão
  • .btn--alternative - Modificador para botão alternative

Formulários

Input - Campo de Texto

Campo de entrada de texto com estados default, focus e error.

Exemplo de Uso

<!-- Input Default -->
<div class="form-group">
    <label for="input-id" class="form-label">Label</label>
    <input type="text" id="input-id" class="form-input" 
           placeholder="Digite seu texto">
</div>

<!-- Input Error -->
<div class="form-group">
    <label for="input-error" class="form-label">Label</label>
    <input type="text" id="input-error" 
           class="form-input form-input--error">
    <span class="form-input-error">Mensagem de erro</span>
</div>
Este campo contém um erro

Classes CSS

  • .form-group - Container do campo de formulário
  • .form-label - Label do campo
  • .form-input - Campo de entrada
  • .form-input--error - Modificador para estado de erro
  • .form-input-error - Mensagem de erro

Checkbox

Componente de checkbox com estados default, checked e disabled.

Exemplo de Uso

<label class="form-checkbox">
    <input type="checkbox" name="checkbox-name">
    <span class="form-checkbox__box"></span>
    <span class="form-checkbox__label">Checkbox Label</span>
</label>

Radio Button

Componente de radio button com estados default, selected e disabled.

Exemplo de Uso

<label class="form-radio">
    <input type="radio" name="radio-group" value="option1">
    <span class="form-radio__circle"></span>
    <span class="form-radio__label">Radio Label</span>
</label>

Tabs

Tabs Horizontal

Componente de tabs dispostas horizontalmente com estados default, hover e active.

Exemplo de Uso

<ul class="tabs tabs--horizontal" role="tablist">
    <li class="tabs__item tabs__item--active" role="presentation">
        <button type="button" class="tabs__button" 
                role="tab" aria-selected="true">
            Tab 1
        </button>
    </li>
    <li class="tabs__item" role="presentation">
        <button type="button" class="tabs__button" 
                role="tab" aria-selected="false">
            Tab 2
        </button>
    </li>
</ul>

Classes CSS

  • .tabs - Container das tabs
  • .tabs--horizontal - Modificador para tabs horizontais
  • .tabs__item - Item individual da tab
  • .tabs__item--active - Modificador para tab ativa
  • .tabs__button - Botão da tab

Acessibilidade

Diretrizes de Acessibilidade

O design system foi desenvolvido seguindo as melhores práticas de acessibilidade (WCAG 2.1).

Contraste de Cores

Todas as combinações de cores do design system atendem aos requisitos mínimos de contraste:

  • Texto normal: mínimo de 4.5:1
  • Texto grande (18px+): mínimo de 3:1
  • Elementos interativos: mínimo de 3:1

ARIA Labels

Todos os componentes interativos incluem atributos ARIA apropriados:

  • Botões: type="button" para botões não-submit
  • Tabs: role="tablist", role="tab", aria-selected
  • Formulários: Labels associados com for e id

Navegação por Teclado

Todos os componentes são totalmente navegáveis por teclado:

  • Tab: Navega entre elementos focáveis
  • Enter/Space: Ativa botões e checkboxes
  • Setas: Navega entre tabs (quando implementado com JavaScript)

Estados de Foco

Todos os elementos interativos possuem estados de foco visíveis usando a cor Action (#7BB9FA).

Checklist de Acessibilidade

  • ✓ Contraste de cores adequado
  • ✓ Labels em todos os campos de formulário
  • ✓ Atributos ARIA apropriados
  • ✓ Navegação por teclado funcional
  • ✓ Estados de foco visíveis
  • ✓ Textos alternativos (quando aplicável)
  • ✓ Estrutura semântica HTML5
