Bem-vindo ao Design System

Este é um sistema de design básico criado como referência para outros projetos. Explore os componentes e estilos disponíveis abaixo.

Tipografia (Typography)

Hierarquia tipográfica do design system. Cada nível possui tamanho de fonte e line-height específicos para garantir legibilidade e consistência visual.

H1 - Heading 1

Heading 1 - Título Principal

Font Size: 48px Line Height: 60px Variable: --font-size-h1 / --line-height-h1
H2 - Heading 2

Heading 2 - Subtítulo Principal

Font Size: 34px Line Height: 50px Variable: --font-size-h2 / --line-height-h2
H3 - Heading 3

Heading 3 - Subtítulo Secundário

Font Size: 28px Line Height: 38px Variable: --font-size-h3 / --line-height-h3
H4 - Heading 4

Heading 4 - Subtítulo Terciário

Paragraph - Parágrafo

Este é um exemplo de parágrafo. O texto de parágrafo é usado para conteúdo principal e deve garantir boa legibilidade. O line-height foi ajustado para proporcionar espaçamento adequado entre as linhas, facilitando a leitura de textos longos.

Font Size: 14px Line Height: 24px Variable: --font-size-paragraph / --line-height-paragraph

Espaçamento (Spacing)

Sistema de espaçamento do design system. Cada tamanho possui uma variável CSS correspondente para uso consistente em todo o projeto. Os espaçamentos são utilizados para margens, paddings e gaps entre elementos.

XS - Extra Small
Value: 5px Variable: --spacing-xs
S - Small
Value: 10px Variable: --spacing-s
M - Medium
Value: 15px Variable: --spacing-m
L - Large
Value: 25px Variable: --spacing-l
XL - Extra Large
Value: 40px Variable: --spacing-xl
XXL - Extra Extra Large
Value: 65px Variable: --spacing-xxl

Botões (Buttons)

Componentes de botão do design system. Cada botão possui estados específicos (default, hover, disabled) para diferentes interações do usuário. Os botões são fundamentais para ações principais e secundárias na interface.

Botão Primary
Botão principal usado para ações primárias. Utiliza a cor Primary do design system.
Class: .btn .btn--primary Background: var(--color-primary) Color: var(--color-white)
Botão Alternative
Botão alternativo usado para ações secundárias. Utiliza a cor Secondary do design system com borda.
Class: .btn .btn--alternative Background: var(--color-secondary) Color: var(--color-black)
